# GitPDF

## How to use

```bash
$ npm install -g gitpdf

$ gitpdf https://github.com/rhiokim/gitpdf/blob/master/README.md
$ gitpdf https://github.com/jquery/jquery/blob/master/README.md --css=./default.css
```
